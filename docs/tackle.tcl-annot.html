tackle.tcl Annotations

Created from src/tackle.tcl
Tackle is a package manager for the Tcl programming language. This file provides a command-line interface, e.g.: 
   $ tackle ls
   tustack
   tcllib
   tcltls
   $ tackle rm tutstack
   Removed tutstack.
   $ tackle ls
   tcllib
   tcltls
